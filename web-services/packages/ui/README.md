# `@repo/ui` — Shared React Components

Internal React component library built on shadcn/ui primitives (Radix + TailwindCSS). Shared across frontend and any future apps.

## Contents

- `src/` — Reusable UI components (Button, Card, Dialog, etc.)
- Built with TailwindCSS and class-variance-authority

## Usage

```ts
import { Button } from "@repo/ui";
```
